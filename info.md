# Sun Tracker

HACS-kompatible Custom-Integration fuer Home Assistant.

- Berechnet den aktuellen Sonnenstand aus dem Home-Assistant-Standort
- Zeigt Sonnenhoehe, Azimut, Sonnenaufgang, Sonnenuntergang und Solarnoon
- Enthaelt eine Lovelace-Karte mit kompletter Sonnenbahn fuer den aktuellen Tag
- Unterstuetzt One-Click-Hinzufuegen als HACS Custom Repository via My Home Assistant
